### Shortcuts
- Open Main Configuration - Press & hold both thumbsticks ('Sprint' & 'Favorites' buttons).
- Open Pipboy Configuration - While Pipboy is open, press & hold right thumbstick ('Favorites' button).
- Can also be opened via FRIK Settings Holotape.

### Body Adjustments
Use [main configuration mode](https://github.com/rollingrock/Fallout-4-VR-Body/wiki/In%E2%80%90Game-Configuration-Guide-(v69)) to fine tune your avatar body and camera position.  
Use selfie mode to see how your body looks in the world.  
Make sure to adjust both in and out of power armor.  

### Pipboy Configuration
Only applicable is using on-wrist Pipboy.  
Check the holo and non-holo Pipboy screen types. Reposition for your comfort.  
Note: Game window must be in-focus for primary hand control to work.  

### Weapon Adjustment Mode
Enter weapon adjustment mode from [main configuration](https://github.com/rollingrock/Fallout-4-VR-Body/wiki/In%E2%80%90Game-Configuration-Guide-(v69)).  
Use to adjust weapon grip position for primary hand, offhand grip position, throwable holding position, and back-of-hand UI offset.  
Vanilla weapons should all be properly positioned (unless VR scale was changed).  
Use to fix new modded weapons positions.

### Advanced Configuration in FRIK.ini
Most configuration is stored in FRIK.ini file. Much more than can be configured via in-game configuration interfaces.  
Enable FRIK.ini auto reloading to have changed loaded live into the running game to experience the effects.  
CAUTION: FRIK.ini auto reload will override in-game configuration changes.  

*Location*: `%USERPROFILE%\Documents\My Games\Fallout4VR\FRIK_Config`
